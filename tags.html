HTML TAGS


HTML tags are the building blocks of a webpage.
Think of them like Lego bricks — each one adds structure or meaning to your content. They’re written inside angle brackets < > and usually come in pairs:
  • an opening tag (<tagname>)
  • a closingtag (</tagname>)
    • Everything in between is the content




    Here:

    • "<p>" is an opening tag.
    • "</p>" is a closing tag.
    • "Hello,world!" is content

    Together, they tell the browser:
    “Hey! Display this text as a paragraph.”


    #Some tags don’t need closing ones — they’re called self-closing or void tags, like:



    Some common HTML tags:

    Tags Meaning Structure
    <h1> to <h6 Headings(h1=biggest,h6 smallest) <h1>title<h1>
    <p> Paragraph <p>Text here<p>
    <a> Link <a href="https://example.com">Click me<a>
    <img> IMage <img src="pic.png" alt="description">
    <div> Division/Block <div>Section<div>
    <ul>,<ol>,<li> Lists <ul><li>Items<li><ul>



    For more HTML Tags:Visit here.

    In-short : tags = the grammar of the web. They don’t do the fancy styling (that’s CSS), nor make things interactive (that’s JS), but they define what everything is.
