In HTML (HyperText Markup Language), colors are used to style elements on a webpage â€” for example, setting the background, text, or border color.

HTML colors can be defined in several ways:


  1. By Name:
    2. HTML recognizes 140 standard color names, like:

    <p style="color: red;">This is red text.</p>
    <p style="background-color: lightblue;">Light blue background</p>


    Some examples of named colors:

    • red
    • blue
    • green
    • lightgray
    • orange
    • purple



  2. By RGB Value:(RGB=Red,Green,Blue)

    3. You can define colors by their Red, Green, Blue components:

    <p style="color: rgb(255, 0, 0);">Red text</p>
    <p style="background-color: rgb(0, 0, 255);">Blue background</p>


    In HTML/CSS, you can mix these colors to make any color you want.

    Each of the three values (red, green, blue) can be:

    • An integer between 0 and 255
    • Or a percentage between 0% and 100%




    ðŸ§© Examples:

    Color RGB code
    Red rgb(255, 0, 0)
    Green rgb(0, 255, 0)
    Blue rgb(0, 0, 255)
    Black rgb(0, 0, 0)
    white rgb(255, 255, 255)
    Gray rgb(128,128,128)
    Yellow rgb(255, 255, 0)
    Magenta rgb(255, 0, 255)
    Cyan rgb(0, 255, 255)





  3. By HEX Code:


    4. A hexadecimal (HEX) color code starts with #, followed by 6 digits (0 to 9, A to F):
    Each pair (RR, GG, BB) is a hexadecimal number from 00 to FF (0â€“255 in decimal).

    <p style="color: #FF0000;">This is red text.</p>
    <p style="background-color: #00FF00;">Green background</p>



    • #FF0000 = red
    • #00FF00 = green
    • #0000FF = blue
    • #FFFFFF = white
    • #000000=black




    ðŸ§© Short HEX Codes:

    If all the pairs are the same (like #FF0000 â†’ #F00), you can shorthand it to 3 digits:
    Example: #0F0=Green




  4. By RGBA Value:
    5. RGBA adds transparency (alpha):

    <p style="background-color: rgba(255, 0, 0, 0.5);"> Semi-transparent red background </p>


    The fourth value (0â€“1) controls opacity:


    • 1 = fully opaque;
    • 0.5 = 50% transparent;
    • 0 = fully transparent;




  5. By HSL / HSLA: [HSLA=Hue, Saturation, Lightness (and Alpha):]

    6. <p style="color: hsl(120, 100%, 50%);">Bright green</p>
    <p style="background-color: hsla(240, 100%, 50%, 0.3);"> Semi-transparent blue </p>
